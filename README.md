# 📦 Inventory Management System (C++)

A console-based inventory management system built using C++ that allows users to manage product records efficiently. The program supports adding products, viewing inventory, and selling items with real-time stock updates.

---

## 🚀 Features

* ➕ Add new products (ID, Name, Price, Quantity)
* 📋 View all products in a structured format
* 💰 Sell products with stock validation
* ⚙️ Menu-driven interface for easy interaction

---

## 🛠️ Technologies Used

* **Language:** C++
* **Concepts:** Object-Oriented Programming (OOP), STL (Vectors)

---

## 🧠 How It Works

* Products are stored using a `vector` of objects
* Each product contains:

  * ID
  * Name
  * Price
  * Quantity
* The system allows users to:

  * Add products to inventory
  * View all stored products
  * Sell products and update stock

---

## ▶️ How to Run

1. Copy the code into a `.cpp` file

2. Compile using a C++ compiler:

   ```
   g++ filename.cpp -o inventory
   ```

3. Run the program:

   ```
   ./inventory
   ```
